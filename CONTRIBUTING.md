# Contributing

Thanks for helping improve this project.

## Quick Checklist

1. Fork the repo and create a feature branch.
2. Add or update a hook in `src/hooks/` and export it from `src/hooks/index.js`.
3. Add or update the matching demo in `src/demos/`.
4. Add or update tests in `src/hooks/__tests__/`.
5. Run `npm run test` and confirm tests pass.
6. Open a pull request with a short summary of what changed and why.

## Scope

- Keep changes focused and small.
- Follow the existing code style and naming patterns.
- Update docs when behavior changes.
