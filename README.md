# React Custom Hooks Interview Prep (JavaScript)

This repo is focused on implementing and practicing common custom hooks that are frequently asked in React interviews.

## Hook List

1. `useCounter`
2. `useToggle`
3. `usePrevious`
4. `useDebounce`
5. `useLocalStorage`
6. `useFetch`
7. `useOnClickOutside`
8. `useWindowSize`
9. `useInterval`
10. `useCopyToClipboard`
11. `useWhyDidItUpdate`

## Project Structure

- `src/hooks`: all custom hook implementations
- `src/App.jsx`: demo UI that imports hooks and verifies behavior manually
- `src/hooks/__tests__`: automated tests for hooks

## Getting Started

```bash
npm install
npm run dev
```

Open the local Vite URL (usually `http://localhost:5173`) to interact with hook demos.

## Run Tests

```bash
npm run test
```

This validates hook logic with unit-style tests.
